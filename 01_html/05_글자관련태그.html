05_글자관련태그

1. b

: bold의 약자로 글자를 진하게 설정하고자 할 때 사용

2. strong

: b태그와 같이 글자를 진하게 하고자 할 때 사용하며, 일반적인 강조는 b태그를, 중요한 단어는 strong태그를 사용

: 시각장애인의 스크린리더 프로그램을 통해 음성이 나올 때 strong태그는 좀 더 강한 어조의 음성이 나옴

3. i

: italic의 약자로 글자를 기울이고자 할 때 사용

4. em

: emphasized의 약자로서 글자를 강조하면서 기울이고자할 때 사용

5. ins

: insert(삽입/수정)의 약자로서 문단내용 중 수정된 글자가 있는 경우 글자에 밑줄로 표시해주는 태그

6. s

: 문단의 내용 중 취소하조가 하는 글자에 취소선 긋기

7. small

: 글자의 원래 크기(16px)보다 3/4크기로 표시하기 위한 태그

8. sup

: superscript의 약자로 위첨자를 나타냄

9. sub

: subscript의 약자로 아래첨자를 나타냄

10.mark

: 문단 내용 중에 사용자가 지정한 범위에 마킹을 한 것처럼 배경색이 노랑색 으로 표시

11. span

: 태그 자체로는 의미가 없지만 단락(p)안에서 일부분의 텍스트만 묶어서 css(디자인서식)을 적용하고자 할 때 사용함

12. address

: 웹사이트 하단의 회사 정보를 입력하기 위한 태그요소로서 글자가 기울게 표시됨.
