09_하이퍼링크

하이퍼링크란?
하이퍼텍스트로 이루어진 문서에서 사용자가 마우스로 클릭을 하였을 때 다른 문서나 페이지로 이동하기 위한 요소로서

1. 텍스트 하이퍼링크,
2. 이미지 하이퍼링크,
3. 이메일 하이퍼링크,
4. 특정문서의 특정위치로 이동하기 위한 링크,
5. 하나의 이미지에 여러개의 링크를 할 수 있는 이미지 맵 링크가 있음

작성법 : anchor hyper reference의 약자인 a태그와 href속성, title 속성을 사용한다


1. 텍스트에 하이퍼링크 걸기

- 바로가기, 자세히보기, 더보기, 로그인하기, 비번찾기 등

- 텍스트 링크는 기본적으로 밑줄, 파랑색으로 표시되며, 한 번 이상 누른적이 있는 경우 보라색으로 변경됨.


네이버 바로가기 다음 바로가기

2. 이미지에 하이퍼링크 걸기


3. 이메일 링크

mailto : 홈페이지, 브랜드 관리자 이메일 연결

관리자에게 메일 보내기

4. 특정 위치로 이동하기 위한 네임앵커

네임앵커란? 특정위치에 id값을 표시한 후 하이퍼링크시 즈소에 아이디값을 설정하면 특정위치로 페이지가 이동하는 기능(책갈피기능)

2. 이미지에 하이퍼링크 걸기
