06_목록태그(list)

: 목록(list)이란? 관련있는 주제에 맞는 내용을 나열하고자 할 때 사용하며,
: 목록관련태그에는 '정의목록', '순차목록', '비순차목록'이 있음


1. 정의목록

: 용어에 대한 정의를 내릴 때 사용하며,
dl(definition list), dt(definition term), dd(definition description)태그로 작성함

dt : 제목
dd : 설명1
dd : 설명2
html이란?
Hypertext markup language의 약자로서 웹문서의 구조를 만들기 위한 언어이다.
문법이 간단하기 때문에 누구나 쉽게 익힐 수 있다.
문서를 작성하고 저장 시 반드시 확장자는 .html 또는 .htm으로 저장한다.
가장 최신버전은 html5이다.

2. 순차목록

: 순차목록은순서가 있는 목록으로서 order list의 약자인ol, li태그 요소를 사용
: type 속성을 사용하여 숫자, 영문 대/소문자, 로마자 대/소문자로 기호를 변경할 수 있음
: 최신뉴스, 영화랭킹, 실시간 음원순위 등에서 사용됨

  1. 순차목록 type="1"
  2. 순차목록 type="1"
  3. 순차목록 type="1"
  1. 순차목록 type="A"
  2. 순차목록 type="A"
  3. 순차목록 type="A"
  1. 순차목록 type="I"
  2. 순차목록 type="I"
  3. 순차목록 type="I"

3. 비순차목록

: 순서가 없는 목록으로 unorder list의 약자인 ul,li태그를 사용하며,
: type 속성을 사용하여 circle(기본값), square모양으로 변경이 가능함
: 웹페이지의 내비게이션 메뉴, 쇼핑몰 메뉴, 이미지 갤러리, 상품 목록, 글 목록 등의 콘텐츠 작성 시 사용하며 가장 많이 사용하는 태그 중 하나임
