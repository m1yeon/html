11_아이프레임

인라인 프레임(아이프레임)이란?
- 문서 안에 또 다른 문서를 삽입하기 위한 태그요소이며
- 회원가입시 나오는 페이지의 약관내용, 회사소개 페이지의 실시간 지도,유튜브의 영상삽입 등
 원하는 페이지를 삽입하여 하나의 문서처럼 보이게 할 수 있음.

- 작성법) iframe="삽입할 문서의 주소나 경로"


iframe 태그 사용시 주의할 점

- 현재는 반응형웹이 주로 대세라서 잘 사용하지 않음

- 페이지가 분리되는 단점이 있음

- 웹접근성에 맞지 않아 자주 사용하지 않음

- 추후 css, javascript, jquery로 해당 콘텐츠 구현이 가능하기 때문에 사용하지 않음

- 최근에는 지도나 영상을 삽입 할 때에만 사용함


1. 현재 문서에 다른 문서 삽입하기

- frame border : 테두리 표시하는 속성

- 개발자 사이트 가입하면 지도에 마크 넣는 거 할 수 있음
